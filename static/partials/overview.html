Overview

A sample application showing Venmurasu Search functionality.

Click 'Search' from the left navigation, enter the word or phrase to search for.

The results should show up with the word/phrase highlighted. Clicking on the Chapter should take you to the corresponding web page of venmurasu.in

The Search query accepts following options currently:

Search by phrase or word like:

  • அர்ஜுனன்
  • "இளைய யாதவர்"
  • "நுரை எழுந்து காத்திருக்கிறது பாலாழி"

    • Search by tags like:

  • tags:அர்ஜுனன்
  • tags:திருஷ்டத்யும்னன்

    • Combined search like:

  • search:'இளைய யாதவர்' tags:அர்ஜுனன்
